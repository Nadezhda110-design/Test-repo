Git

  • git init - инициализируем репозиторий
  • dir /ah - проверить наличие скрытых файлов.
  • git status - проверить изменения в удаленном репозитории.
  • git add - добавить файл в отслеживаемые файлы.
  • git commit -m "first commit" - сделать коммит.
  • git push - внести изменения на удаленный репозиторий.
  • git log - проверить изменения на удаленном репозитории
